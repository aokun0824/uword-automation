# ユーワード自動投稿サービス 利用マニュアル

---

## はじめに

このサービスは、**ユーワードへの投稿を毎日自動で行う**ツールです。
AIが最新のニュースをもとに、あなたのお店・サービスに合った投稿文を自動生成して投稿します。

設定はブラウザから簡単に変更できます。難しい操作は一切不要です。

---

## 1. 管理画面へのアクセス

ブラウザ（Chrome・Safari など）で以下のURLを開いてください。

```
https://uword-automation.onrender.com
```

> **注意:** 一定時間アクセスがないと画面が表示されるまで最大1分ほどかかることがあります。
> 少し待ってからページを再読み込みしてください。

---

## 2. 新規登録

初めて利用する場合は、ログイン画面の「新規登録はこちら」から登録してください。

### 入力項目

**ユーワードのログイン情報**

| 項目 | 説明 |
|------|------|
| 会員番号 | ユーワードの会員番号（例: 83900）。管理画面のログインIDになります |
| ユーワード ログインID | ユーワードにログインする際のID |
| ユーワード パスワード | ユーワードにログインする際のパスワード。管理画面のログインにも使われます |
| ユーザーパス | ユーワードのマイページURLに含まれるパス（例: `u-word.com/`**horby**`/myPage` の太字部分） |

**お店・サービスの情報**

| 項目 | 説明 |
|------|------|
| 業種 | 6種類から選択。選んだ業種に合ったニュースをもとに投稿文が生成されます |
| 店舗・サービス名 | 投稿に使われる名前 |
| サービス説明 | どんなサービスか（AIが内容を合わせるために使います） |
| 締め文（CTA） | 投稿の最後に自動でつく一言（例: お気軽にご相談ください） |

---

## 3. ログイン

| 項目 | 内容 |
|------|------|
| ユーザー名 | 会員番号（例: `83900`） |
| パスワード | ユーワードのパスワード |

---

## 4. 設定画面の見かた

ログインすると設定画面が開きます。以下の項目を変更できます。

---

### プロフィール

お店・サービスの基本情報です。投稿の内容やトーンに影響します。

| 項目 | 説明 |
|------|------|
| 店舗・サービス名 | 投稿に使われる名前 |
| サービス説明 | どんなサービスか |
| 締め文（CTA） | 投稿の最後に自動でつく一言 |

---

### 投稿スケジュール

何時に自動投稿するかを確認できます。

> 投稿時刻の変更は担当者にご相談ください。

---

### 投稿ネタのジャンル

6種類の業種から選ぶだけで、関連するニュースが自動で取得されます。

| 選択肢 |
|--------|
| AI・IT・デジタル |
| 美容・サロン・エステ |
| 飲食・グルメ・カフェ |
| 健康・医療・整体 |
| 教育・スクール・講座 |
| その他 |

---

### アピールしたいキーワード

投稿に必ず絡めてほしいキーワードを最大3つ入力できます。

**入力例:**
- 初心者向け
- AI活用
- 無料相談

---

### 紹介したいメニュー・サービス

お店のメニューやサービス名を登録しておくと、AIが毎回1つ選んで投稿に自然にPRしてくれます。

**入力例（1行に1件）:**
```
HP制作（5万円〜）
AIチャットボット導入支援
無料オンライン相談
```

---

### 次回投稿の文章を指定する

この機能を使うと、**自分で書いた文章を指定した時間に投稿**できます。

1. タイトルと本文を入力して保存
2. 次の投稿時刻（朝9時または夜9時）に、その文章が自動投稿される
3. 投稿後は自動でクリアされ、以降はAI生成に戻る

> お知らせ・キャンペーン情報・季節の挨拶などに便利です。

---

### 投稿の文体・トーン

AIへの指示です。ブランドの雰囲気に合わせてカスタマイズできます（1行に1件）。

---

## 5. 設定を保存する

変更が終わったら、画面下部の「**設定を保存**」ボタンをクリックしてください。
「設定を保存しました」と表示されたら完了です。次回の投稿から新しい設定が反映されます。

---

## 6. 投稿履歴を確認する

画面右上の「**投稿履歴**」ボタンから、過去の投稿内容を確認できます。

---

## よくある質問

**Q. 投稿が止まっているようです**
A. ユーワードのログインID・パスワードが変わっていないか確認してください。変更があった場合は担当者にご連絡ください。

**Q. 投稿時刻を変えたい**
A. 担当者にご連絡ください。

**Q. パスワードを忘れました**
A. 担当者にご連絡いただければ再設定します。

**Q. ログイン画面が表示されるまで時間がかかります**
A. 無料プランのため、しばらく使われていないとシステムが一時停止します。最大1分ほどお待ちいただくと表示されます。

**Q. 自分で書いた文章はいつ投稿されますか？**
A. 保存後、次の投稿時刻（朝9時または夜9時）に投稿されます。投稿後は自動でクリアされます。
